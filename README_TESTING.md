# Agent Modpack (TESTING)

> **Warning: This is an unstable development build.** It is published automatically on every merge to `main` and is not intended for casual play. Mod lists, configs, and dependencies may change without notice and can break existing saves or multiplayer sessions.
>
> **Use the stable modpack instead:** [Agent_Modpack on Thunderstore](https://thunderstore.io/c/lethal-company/p/joshhsoj1902/Agent_Modpack/)

## About this package

This Thunderstore package (`Agent_Modpack_TESTING`) tracks the latest `main` branch. Versions are auto-published as `1.0.<build>` where `<build>` is the GitHub Actions run number, so they update frequently.

For the full mod list, config notes, and project description, see the [README on GitHub](https://github.com/joshhsoj1902/lethal-company-agent-modpack/blob/main/README.md).
